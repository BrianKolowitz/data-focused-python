## for loops


```python
# for loops
for i in range(1, 10, 2):
    print(i)
```

    1
    3
    5
    7
    9



```python
# for...else loop
for i in range(1, 5):
    print(i)
else:
    print('printed all items')
```

    1
    2
    3
    4
    printed all items



```python
# for...else loop with a break
for i in range(1, 5):
    print(i)
    if i > 2:
        break
else:
    print('printed all items')
```

    1
    2
    3


## while loops


```python
# while loop
condition = True
i = 0
while condition:
    print(i)
    i += 1
    if i > 3:
        condition = False
```

    0
    1
    2
    3

## nested loops


```python
width = 5
chars = "*@"
for char in chars:
    for i in range(1, width):
        print(char * i)
    for j in range(width, 0, -1):
        print(char * j)
```
